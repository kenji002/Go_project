# Go Project

Go言語の基本操作（変数、型、パッケージ）を学ぶための学習用プロジェクトです。

## プロジェクト構成

```text
Go_project/
├── go.mod          # モジュール定義
├── say01.go        # メインプログラム
├── utils/          # ユーティリティパッケージ
│   └── utils.go    # 共通関数（加算など）
└── README.md       # 本ドキュメント
```

## 動作確認済み環境

- Go 1.25.0 以上

## 実行方法

ターミナルでプロジェクトのルートディレクトリに移動し、以下のコマンドを実行してください。

```bash
go run say01.go
```

または、ディレクトリ内のすべてのファイルを実行対象とする場合は以下でも実行可能です。

```bash
go run .
```

## 主な機能

### `say01.go`
- 変数の宣言方法（明示的型宣言と短縮宣言）のデモンストレーション
- 基本的な型（int, string, float64, bool）の操作
- `reflect` パッケージを使用した型情報の取得
- 外部パッケージ（`utils`）の関数呼び出し

### `utils` パッケージ
- `Add`: 2つの整数の和を返す関数
- `PrintAdd`: 加算結果をターミナルに表示する関数
